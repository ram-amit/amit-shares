Geo model holdout validation: YT Brand US V2 design, QES

Real data: QES by DMA from MARKETING.L3.FACT_GEO_KPI_DAILY, Jan 1 - Jul 15 2026. Geo groups are the actual YT Brand US V2 design (21 treatment / 71 control DMAs). The test never launched, so the whole window is treatment-free and every holdout is clean.

warn - Out-of-sample accuracy is borderline; review before trusting test readouts. Worst-fold MAPE 13.6%, worst mean bias 4.8%, lowest prediction-interval coverage 93% across 4 fold(s). Thresholds are placeholders pending a DS-approved standard.
Method. The production TBR model (OLS of treatment daily sum on control daily sum, weekday-trained, outliers removed via modified z-score - ported line-for-line from bb-machine-learning geo_tool) is trained only on the window left of the red dashed line, then predicts the holdout window it has never seen. Metrics compare those predictions with reality. This is the clean-test-set validation the production pipeline currently skips: in production the model trains on 100% of pretest and is judged only by in-sample fit statistics.
